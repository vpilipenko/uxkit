### Usage example
```js
<Tabs
  name='tabs'
  value={state.tabs}
  onChange={this.handleChange}
>
  <Pane label='Администрирование'>
    Контент для администратора
  </Pane>
  <Pane label='Диспечеризация'>
    Контент для диспечера
  </Pane>
</Tabs>
```
